Technical Addendum

A Comparative Analysis of Multi-Model Binary Classification in High-Stakes Disaster Recovery

Yonas T. A.

Machine Learning Project

This project presents a parallel inference framework for survival prediction using the historical Titanic passenger dataset. The system implements a multi-model architecture referred to as Lifeboat Gates, in which multiple machine learning models operate simultaneously on identical input features to generate independent probabilistic survival estimates.

Rather than producing a single authoritative prediction, the system is designed as a comparative analytical structure in which each model functions as an independent inference agent. This architecture enables direct observation of how different mathematical and computational paradigms interpret the same historical data, transforming prediction into a structured analysis of algorithmic reasoning.

The Logistic Gate serves as the statistical baseline model, applying a sigmoid transformation over a linear combination of features to produce interpretable survival probabilities. It provides coefficient-level transparency and represents classical statistical learning through proportional feature influence and linear decision structure.

The Random Forest and XGBoost gates represent ensemble-based and boosting-based learning systems that model complex non-linear relationships and hierarchical feature interactions. These models capture structured dependencies between class, age, gender, and socioeconomic indicators that are not expressible within linear model assumptions.

The SVM gate introduces margin-based geometric reasoning through decision boundary optimization in high-dimensional feature space, offering a fundamentally different representation of survivability structure that is not probability-driven but separation-driven.

The objective of this project is to construct a transparent multi-model inference system that enables comparative reasoning, interpretability through model disagreement, and structural understanding of inductive bias in machine learning systems. The framework transforms historical disaster data into an experimental platform for studying how different algorithmic systems encode reality, uncertainty, and decision logic.
